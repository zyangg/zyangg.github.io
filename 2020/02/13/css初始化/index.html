css初始化

发表于 2020-02-13 更新于 2020-05-07 分类于 前端 阅读次数： Valine：
本文字数： 1k 阅读时长 ≈ 1 分钟

目的

消除不同浏览器对HTML文本呈现的差异.

一般标签在浏览器中都有默认样式，例如body标签有默认的外边距，ul有默认的小黑点和内边距，前端程序员在写页面的时候会把这些默认的样式都清除掉，让所有标签的外观效果在所有浏览器表现一致，这个步骤就是css的初始化。
